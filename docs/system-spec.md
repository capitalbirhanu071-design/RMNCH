# RMNCH Quantification System Specification

## Overview
Digital platform to support national RMNCH quantification including forecasting, stock analysis, pipeline monitoring, net requirement calculation, supply planning and costing.

## Modules
- Planning Setup
- Product Master
- Forecasting
- Stock Status
- Pipeline
- Net Requirement
- Supply Planning
- Costing
- Reports

## Technology Stack
Frontend: React
Backend: .NET Core API
Database: PostgreSQL
