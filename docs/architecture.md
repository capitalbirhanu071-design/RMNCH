# RMNCH Quantification Tool Architecture

## Overview
Digital RMNCH Quantification and Supply Planning System supporting forecasting, reconciliation, costing, and procurement planning for national health programs.

## Modules

1. Planning Setup
2. Product Master List
3. Data Sources
4. Forecasting
5. Stock Status
6. Pipeline
7. Net Requirement
8. Supply Planning
9. Costing
10. Review and Approval
11. Reports and Dashboards

## Technology Stack

Frontend: React  
Backend: .NET Core API  
Database: PostgreSQL  

## Users

National Program Managers  
Regional Health Bureaus  
EPSS  
Health Facilities  
Partners and Donors
